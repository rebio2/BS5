¿Qué es el Modo Oscuro?

El modo oscuro es una característica visual que utiliza colores oscuros para reducir el brillo de la pantalla y mejorar la experiencia visual en entornos con poca luz. En Bootstrap 5, puedes habilitarlo de manera sencilla y flexible.

Alternar Modo Oscuro

Puedes usar JavaScript para alternar entre los modos claro y oscuro:

document.querySelector('#darkModeToggle').addEventListener('click', () => {
    document.body.classList.toggle('bg-dark');
    document.body.classList.toggle('text-white');
});

Clases Útiles para el Modo Oscuro

  • bg-dark: Fondo oscuro.
  • text-white: Texto blanco.
  • btn-dark: Botón con estilo oscuro.
  • btn-light: Botón con estilo claro.
