¿Qué es Offcanvas?

Offcanvas es un componente de Bootstrap 5 que permite crear menús y paneles deslizantes desde los laterales de la página, ofreciendo una experiencia interactiva para los usuarios. A continuación, exploraremos cómo implementarlo.

1. Estructura Básica del Offcanvas

La estructura básica de un offcanvas es la siguiente:


<div class="offcanvas offcanvas-start" tabindex="-1" id="offcanvasExample" aria-labelledby="offcanvasExampleLabel">
    <div class="offcanvas-header">
        <h5 class="offcanvas-title" id="offcanvasExampleLabel">Offcanvas Título</h5>
        <button type="button" class="btn-close" data-bs-dismiss="offcanvas" aria-label="Close"></button>
    </div>
    <div class="offcanvas-body">
        Aquí puedes colocar contenido como formularios, enlaces o cualquier otra cosa que quieras mostrar en el offcanvas.
    </div>
</div>

En este ejemplo, el offcanvas se despliega desde la izquierda. Puedes cambiar la dirección del offcanvas utilizando diferentes clases como offcanvas-end, offcanvas-top, o offcanvas-bottom.

2. Activando el Offcanvas

El offcanvas se activa a través de un botón o enlace. A continuación, un ejemplo de un botón que abre el offcanvas:


<button class="btn btn-primary" data-bs-toggle="offcanvas" data-bs-target="#offcanvasExample" aria-controls="offcanvasExample">
    Abrir Offcanvas</button>

3. Personalización del Offcanvas

El offcanvas es completamente personalizable. Puedes agregar animaciones, colores y otros estilos para adaptarlo a tu diseño.
