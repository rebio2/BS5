¿Qué es Scrollspy?

El componente Scrollspy de Bootstrap 5 es una característica que permite mantener la navegación sincronizada con el contenido visible en la pantalla, resaltando el enlace del menú de navegación correspondiente a la sección visible.

Explicación del Código

Para implementar Scrollspy en Bootstrap 5, utilizamos el atributo data-bs-spy="scroll" en un elemento contenedor, lo que permite que Bootstrap detecte el desplazamiento de la página y actualice la navegación. El atributo data-bs-target se usa para vincular el contenedor con el navbar correspondiente, y data-bs-offset define el offset para el activador del scrollspy.
