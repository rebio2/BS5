Explicación del Código

A continuación, se explica cómo funciona el código de la barra de navegación (navbar) en Bootstrap 5:

1. Estructura Básica del HTML

El código HTML comienza con la estructura básica de una página:


<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Navbar en Bootstrap 5</title>
        
        <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
        
        <link href="../css/main.css" rel="stylesheet">
    </head>
    <body>
        
    </body>
</html>

Esta es la estructura base que necesitas. Dentro de la etiqueta <head>, se incluyen los enlaces para el CSS de Bootstrap y otros archivos adicionales que puedas necesitar.

2. Barra de Navegación

La barra de navegación está dentro de la etiqueta <nav>. Este es el código básico para crear un navbar:


<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNavDropdown">
            <span class="navbar-toggler-icon"></span>
        </button>

        <div class="collapse navbar-collapse" id="navbarNavDropdown">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link active" href="../index.html">Inicio</a>
                </li>
                
            </ul>
        </div>
    </div>
</nav>

El navbar se define con la clase navbar y se hace responsivo gracias a la clase navbar-expand-lg. Esto significa que el navbar se expandirá en pantallas grandes, pero en pantallas pequeñas, se colapsará y será accesible mediante un botón de hamburguesa.

3. Explicación del Menú Desplegable

El menú desplegable se activa con el botón de hamburguesa, que se ve en pantallas pequeñas. Este botón se define con la clase navbar-toggler y usa data-bs-toggle="collapse" para mostrar u ocultar el menú:


<button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNavDropdown">
    <span class="navbar-toggler-icon"></span>
</button>

El menú colapsable está contenido dentro de <div class="collapse navbar-collapse"> y sus elementos están dentro de una lista desordenada (<ul>) con la clase navbar-nav.

4. Enlaces de Navegación

Cada enlace dentro del navbar se coloca dentro de un <li> con la clase nav-item, y se puede personalizar con la clase nav-link para aplicar los estilos de Bootstrap:


<li class="nav-item">
    <a class="nav-link active" href="../index.html">Inicio</a>
</li>

5. Scripts para Funcionalidad

Finalmente, debes incluir los archivos de JavaScript de Bootstrap para que el menú desplegable funcione correctamente en dispositivos móviles. Los scripts se colocan antes de cerrar la etiqueta </body>:


<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.11.6/dist/umd/popper.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/js/bootstrap.min.js"></script>
