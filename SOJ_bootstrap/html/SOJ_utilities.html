Utilidades de Espaciado (Spacing Utilities)

Bootstrap 5 ofrece clases para controlar el espaciado entre los elementos. Las clases de espaciado permiten agregar márgenes y rellenos con diferentes valores.

Utilidades de Colores (Color Utilities)

Bootstrap 5 incluye una gran variedad de clases para colores de fondo y texto. Estas clases te permiten personalizar rápidamente el color de los elementos.
